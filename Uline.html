Uline UX

Project highlights from my time at Uline working as a UX Designer.

I've learned a lot during my time at Uline. Some of my more impactful projects include redesigning the careers website, building out a design system for the sales application, and updating the registration process for new customers.

Jobs Redesign

OVERVIEW

Updating the look and feel of Uline.jobs to be more user-friendly.

  • Improve the mobile experience.

  • Make relevant information easier to find.

  • Modernize the site with the company's current brand.

Sales Design System

OVERVIEW

Creating a system to streamline concepting and development for Uline's internal Sales tool.

  • Document common design elements and code.

  • Collaborate with development teams.

  • Identify and fix accessibility issues.

Registration

OVERVIEW

Simplifying the registration process on Uline.com.

  • Remove unneccessary fields.

  • Standardize the experience across desktop and mobile.

  • Lay groundwork for implementing Guest Checkout.
